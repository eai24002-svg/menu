# Spirito Vita - QR Menu

قائمة طعام إلكترونية احترافية لمطعم **روح الحياة (Spirito Vita)**

## المميزات

- 📱 تطبيق PWA يفتح كتطبيق على الهاتف (QR / NFC)
- ✨ شاشة تحميل احترافية مع اللوجو
- 🛒 سلة طلبات مع إرسال مباشر لواتساب المطعم
- 🔍 بحث في القائمة
- 👨‍💼 لوحة تحكم للأدمن (إضافة / تعديل / حذف / إخفاء الأطباق)
- 🎨 تصميم بألوان الهوية البصرية للمطعم

## التشغيل

```bash
npm install
npm run dev
```

- **القائمة:** http://localhost:3000
- **لوحة التحكم:** http://localhost:3000/admin
- **كلمة مرور الأدمن الافتراضية:** `spirito2024`

## تغيير كلمة المرور

عدّل ملف `.env.local`:

```
ADMIN_PASSWORD=your_password_here
```

## النشر

```bash
npm run build
npm start
```

## QR Code

بعد النشر، أنشئ QR Code يشير إلى رابط الموقع. عند مسحه يفتح التطبيق مباشرة.
